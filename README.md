# Desktop Sticky Note

一个基于 Electron 的桌面便签应用，支持普通编辑窗口和迷你悬浮条两种模式。项目使用单页 `index.html` 承载界面与交互逻辑，适合本地快速记录笔记、管理待办事项，以及打包成 Windows 桌面应用使用。

## 项目特点

- 双模式切换：完整模式用于编辑和管理，迷你模式用于常驻桌面快速查看
- 便签与待办分离：支持富文本便签和 To-Do 列表两类内容
- 日历筛选：可按日期查看当日创建的内容
- 置顶与搜索：支持固定重要内容、全文搜索名称和正文
- 颜色自定义：便签和待办列表都可以单独设置颜色
- 图片粘贴：便签编辑区支持直接粘贴图片
- 本地持久化：数据保存在浏览器环境的 `localStorage`
- 导出能力：当前便签或待办列表可导出为 `txt` 或 `md`
- Windows 打包：已配置 `electron-builder`，可生成安装包和便携版

## 技术栈

## 目录结构

```text

├─ index.html          # 主界面，包含样式和前端逻辑

├─ preload.js          # 暴露安全的渲染进程 API
├─ package.json        # 项目配置与脚本
├─ package-lock.json
├─ dist/               # 打包输出目录
```

## 运行方式

先安装依赖：

```bash
npm install
```

启动开发版本：

```bash
npm start
```

也可以使用：

```bash
npm run dev
```

## 打包命令

打包当前平台：

```bash
npm run dist
```

仅打包 Windows：

```bash
npm run dist:win
```

仅生成未压缩目录：

```bash
npm run pack
```

打包产物默认输出到 `dist/` 目录。

## 核心功能说明

### 1. 完整模式

- 左侧边栏显示日历、便签列表、待办列表
- 主区域根据当前选中项显示便签编辑器或待办列表
- 顶部工具栏支持新建、搜索、置顶、改色、删除、导出、切换迷你模式

### 2. 迷你模式

- 窗口缩小为窄条形悬浮视图
- 窗口会保持置顶显示
- 可快速预览便签内容
- 可直接勾选待办项
- 可从迷你模式一键切回完整模式

### 3. 数据存储

应用不依赖后端，数据默认保存在本地：

- `stickyapp_data`：便签和待办的主体数据
- `stickyapp_ui`：当前激活项、筛选条件、迷你模式位置等界面状态

这意味着：

- 重启应用后数据仍会保留
- 更换设备或清空本地存储后数据不会自动同步

## 主要文件职责

### `electron-main.js`

- 创建主窗口
- 处理完整模式与迷你模式切换
- 控制窗口尺寸、是否置顶、是否可缩放
- 监听窗口移动并同步迷你模式位置

### `preload.js`

- 通过 `contextBridge` 向前端暴露 `desktopShell`
- 提供 `switchToMini`、`switchToFull`、`resizeMini` 等 IPC 能力

### `index.html`

- 包含界面结构、样式定义和主要业务逻辑
- 管理便签、待办、搜索、筛选、导出、颜色选择、图片粘贴等功能
